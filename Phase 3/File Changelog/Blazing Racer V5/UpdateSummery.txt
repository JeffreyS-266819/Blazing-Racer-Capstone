In version 5, The Sign In Screen was made in a separate Unity project in this version it was merged with the main project. Also the model car was broken, specifically when the FBX file was exported it was making the meshes scaled to 100 which would cause to not be able to move especially up slopes. That was fixed by changing the 'Apply Scalings' option from 'All Local' to 'FBX Unit Scales' in the exporting menu.
